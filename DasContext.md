Retrieves the DNA of a mutation using a DAShttp://www.biodas.org server.

# Parameters #


  * **DAS DNA URL** : DAS DNA URL. example for UCSC hg19:
```
http://genome.ucsc.edu/cgi-bin/das/hg19/dna
```

  * **CHROM** : VCF CHROM
  * **POS** : VCF Position
  * **extend** : number of bases on each side
