this node parses SIFThttp://sift.jcvi.org/www/SIFT_chr_coords_submit.html results

# Parameters #


  * **CHROM** : VCF CHROM
  * **POS** : VCF POS
  * **REF** : VCF REF
  * **ALT** : VCF ALT
  * **SIFT** : Sift column containing the mutation

# Input #


  * **in1** : VCF

  * **int2** : SIFT result


# Output #


  * **out1** : VCF
